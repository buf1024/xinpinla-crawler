奇妙清单

By buf1024

8月 1 2014 更新日期:8月 1 2014

文章目录

Wunderlist 作为 iOS 平台的老牌 GTD 类工具对用户而言并不陌生。他们曾在自己的博客中暗示，在较长的一段沉默后，第三版 Wunderlist 会有大改变。

今天 Wunderlist 3 终于发布，除 UI 做出重大改进外还主推清单分享和实时同步功能。比如，你可以和家人共享超市购物清单以便确认和随时补充，也可以和旅游同行的好友确认必备品和行进路线。在社交外，不难发现 Wunderlist 3 其实暗含着进军中小团队协作工具的目标。新版 Wunderlist 可以实现清单分发和内容实时同步，仪仗它跨多平台的优势让所有团队成员可以在第一时间获知任务动态并以清单形式交流信息。

此外，这次 Wunderlist 正式进入中国市场。它的中文名叫“奇妙清单”。在程序本体的汉化工作之外，用户还可以通过微博、微信、QQ 等国内社交、通讯方式分享创建的清单。分享后的清单可被其他用户直接加入自己的账户清单列表中。

奇妙清单首发平台有 iPhone、iPad、Mac、Android、Kindle Fire、Chrome OS （Chrome 浏览器插件）和网页版，Windows 8 商店版和 Windows Phone 8 版本将在随后推出。目前国内用户登录和注册“奇妙清单”的速度仍有些缓慢，希望他们能加快服务器或 CDN 落地，真正方便国内用户使用。

感兴趣的用户可以在奇妙清单官网注册账户并下载客户端使用。
